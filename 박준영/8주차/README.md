# 핵심 키워드

- java의 Exception 종류들

    | 구분 | 설명 | 대표 예시 | 처리 필요 여부 |
    | --- | --- | --- | --- |
    | **Checked Exception** | **컴파일 시점에 반드시 처리해야 하는 예외** | `IOException` | try-catch or throws 필수 |
    | **Unchecked Exception (Runtime Exception)** | **실행 중 발생하는 예외**(프로그래머의 실수 등) | `NullPointerException` | 명시적 처리 선택 |
